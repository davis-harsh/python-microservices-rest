# Python Microservices with REST APIs

A distributed microservices architecture built with Python, demonstrating async communication, error handling, and RESTful design.

## 🔧 Features
- Multiple independent microservices
- REST API communication
- Async processing
- Structured error handling
- JSON data interchange

## 🛠 Technologies
Python  
Flask  
Docker  
Postman for API testing  

## 📘 Architecture
- Service A: Data ingestion  
- Service B: Processing  
- Service C: Aggregation  

## 🧪 Testing
Load testing and API validation using Postman collections.
